# Dilshuppa_IDS

A simple Intrusion Detection System (IDS) for detecting network attacks using Scapy.

## Features

- Detects SYN Scan
- Detects Denial of Service (DoS) attacks
- Detects Xmas Scan
- Detects NULL Scan
- Detects FIN Scan
- Detects ARP Spoofing/Poisoning
- Detects Port Scanning
